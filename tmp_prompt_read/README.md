# PACZKA DO CODEX — Zakładka Kontakt, kierunek 1: zdjęcie

Projekt: regulskibehawiorysta.pl / Behawior 15  
Cel: wdrożyć kierunek oparty o zdjęcie na podstronie Kontakt, bez przebudowy całego serwisu.

## Efekt końcowy

Zakładka Kontakt ma dostać spokojną, autentyczną fotografię behawiorysty / konsultanta w rozmowie z opiekunem psa. Zdjęcie ma ocieplać formularz, budować zaufanie i obniżać napięcie przed wysłaniem wiadomości.

Nie robimy nowej strony od zera. Wzmacniamy obecny layout.

## Najważniejsze założenie

Obecna strona ma jasny, elegancki, beżowo-brązowy styl. Zdjęcie nie może wyglądać jak tani stock, reklama psiej szkoły ani weterynaryjna grafika. Ma wyglądać jak dokumentalna scena konsultacji: spokojna rozmowa, pies obok, naturalne światło, dużo przestrzeni.

## Pliki dla Codex

1. `CODEX_PROMPT.md` — główny prompt do wklejenia w Codex.
2. `VISUAL_SPEC.md` — specyfikacja wizualna i UX.
3. `IMAGE_PROMPTS_FLUX.md` — prompty do wygenerowania zdjęcia.
4. `ACCEPTANCE_CRITERIA.md` — checklista odbioru.
5. `IMPLEMENTATION_NOTES.md` — wskazówki techniczne.
