📦 Paczka #3 — Opinie + Google Reviews

Sekcja social proof — karuzela opinii, integracja z Google Business Profile, Schema.org Review markup (gwiazdki w SERP).

Strategia hybrydowa: komponent najpierw próbuje pobrać live opinie z Google API (24h cache), w razie błędu/braku klucza spada na manualną listę z reviews.config.ts. Schema.org działa w obu wariantach.

🔄 Flow danych

Google Places API ──► /api/reviews ──► <ReviewsSection> ▲ │ │ (jeśli błąd lub brak klucza) ▼ reviews.config.ts ────────────► Schema.org JSON-LD │ ▼ Google indeksuje │ ▼ ⭐ gwiazdki w SERP

✨ Co dostajesz

🎠 Karuzela opinii
Auto-scroll co 6s, swipe na mobile, strzałki ←→ z klawiatury, kropki nawigacji, pauza on hover.
⭐ Schema.org markup
JSON-LD typu LocalBusiness + AggregateRating + Review. Po indeksacji → gwiazdki w wynikach Google.
🔍 Google Places API
Live pull opinii z Google Business Profile, cache 24h, automatyczne anonimizowanie imion (RODO).
📋 Manualny fallback
8 przykładowych opinii do wymiany. Działa bez Google API. Pełna kontrola nad treścią.
🏆 Google Rating Badge
"4.9 ⭐ z Google" — wbudowane logo Google jako SVG, klikalne, prowadzi do profilu.
📱 Responsywne
Karuzela 1 kolumna mobile, grid 3 kolumny desktop. Touch-friendly, klawiaturo-friendly.

🚀 Wdrożenie

  1. Pobierz folder handoff-3/ i wgraj do roota repo
  2. Otwórz Claude CLI w katalogu repo
  3. Wklej prompt z PROMPT-FOR-CLAUDE-CLI.md
  4. Edytuj reviews.config.ts — zastąp przykładowe opinie prawdziwymi
  5. Build + deploy → opinie + gwiazdki w SERP
  6. (Opcjonalnie) Skonfiguruj Google API dla live opinii — instrukcja w README

💡 Best practices dla behawiorystyki

  • Wybieraj opinie konkretne, nie ogólne. „Pies przestał ciągnąć po 2 sesjach" > „Polecam!"
  • Pokaż różne problemy — reaktywność, lęk separacyjny, konflikt kotów, szczeniak.
  • Inicjały + miasto zamiast pełnych imion (RODO + dyskrecja klientów).
  • Cytat + nazwa zwierzaka — „Marta K. (właścicielka Burka)" — ludzkie, ciepłe.
  • Nie wszystkie 5⭐. Czasami 4⭐ z konstruktywną opinią buduje wiarygodność.
Po deployu — sprawdź Schema.org:
Wejdź na Google Rich Results Test i wpisz regulskibehawiorysta.pl. Powinieneś zobaczyć:
  • LocalBusiness z aggregateRating
  • Listę Review (8+ opinii)
  • Brak błędów / ostrzeżeń
Po 1-2 tygodniach od ponownej indeksacji → gwiazdki w wynikach Google.
