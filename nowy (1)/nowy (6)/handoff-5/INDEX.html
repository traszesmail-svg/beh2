📦 Paczka #5 — Dark mode

Pełny przełącznik dzień/noc. Respektuje OS, zapisuje wybór, zero "flash" przy odświeżeniu.

Stack: next-themes (3KB) + Tailwind darkMode: 'class' + CSS custom properties. Standard branżowy. SSR-safe.

🔘 Trzy warianty przełącznika

  • variant="icon" — pojedyncza ikona ☀️/🌙, toggluje light↔dark. Najmniej miejsca, najlepszy do nawigacji.
  • variant="segmented" — segmented control: ☀️ Jasny | 💻 System | 🌙 Ciemny. Pełna kontrola, więcej miejsca.
  • variant="dropdown" — ikona z menu hover. Dyskretny, dobry dla minimalistycznych nawigacji.

🚀 Wdrożenie

  1. Pobierz folder handoff-5/ i wgraj do roota repo
  2. Otwórz Claude CLI w katalogu repo
  3. Wklej prompt z PROMPT-FOR-CLAUDE-CLI.md
  4. Sprawdź każdą stronę w obu trybach (lista w PROMPT)
  5. Deploy
⚠️ Co potrzebuje uwagi po wdrożeniu:
  • Ilustracje Storyset (paczka #2) mają jasne tło — w dark mode przyciemnić filtrem lub przygotować osobne wersje
  • Logo — jeśli czarne, podmień na białą wersję w dark
  • Hard-coded kolory w komponentach — wyszukaj bg-white, #fff, text-black i zamień na CSS vars
  • Kontrast — sprawdź Lighthouse a11y w obu trybach (cel: 100/100)
