📦 Paczka #6 — SEO + Open Graph

Pełna optymalizacja SEO. Meta tags per strona, dynamiczne OG images, structured data, sitemap, robots.txt, breadcrumbs.

Stack: Next.js Metadata API + native sitemap.ts/robots.ts + next/og dla dynamicznych obrazków + Schema.org JSON-LD. Zero zewnętrznych zależności.

✨ Co dostajesz

📋 Meta tags per strona
15+ stron, każda z unikalnym title (≤60 znaków), description (≤160), keywords, canonical URL, robots.
🖼️ Open Graph + Twitter
Dynamicznie generowane obrazki 1200×630 dla każdej strony. Tytuł, gradient, badges.
🏷️ Structured Data
LocalBusiness, Service, FAQPage, Article, Person, BreadcrumbList, AggregateRating.
🗺️ Sitemap + Robots
Auto-generowane przez Next.js. Z priorytetami i częstotliwością. Ready do submit w GSC.
🍞 Breadcrumbs
Komponent UI + JSON-LD. Pokazuje się w wynikach Google + ułatwia nawigację.
📱 PWA Manifest
manifest.webmanifest — instalowalne na telefonie. Theme color, icons, standalone display.

🚀 Wdrożenie

  1. Pobierz folder handoff-6/ i wgraj do roota repo
  2. Otwórz Claude CLI w katalogu repo
  3. Wklej prompt z PROMPT-FOR-CLAUDE-CLI.md
  4. Edytuj seo.config.ts jeśli chcesz dostosować titles/descriptions
  5. Sprawdź favicony w /public/ — wygeneruj z logo jeśli brak
  6. Build + deploy
  7. Submit sitemap w Google Search Console
📊 Po deployu — test:
  • Google Rich Results Test — Schema.org
  • OpenGraph.xyz — podgląd FB / Twitter / LinkedIn
  • PageSpeed Insights — Core Web Vitals
  • Google Search Console — submit sitemap, monitoring
