🚀 Wdrożenie

  1. Otwórz Claude CLI w katalogu repo
  2. Wgraj folder handoff-2/ do roota
  3. Wklej prompt z PROMPT-FOR-CLAUDE-CLI.md
  4. Pobierz ilustracje ze Storyset (14 plików SVG zgodnie z mapą wyżej)
  5. Wrzuć do public/illustrations/ z nazwami z config-a
  6. Build + deploy
📝 Storyset — ustawienia kolorów:
  • Style: Pana (line-art z lekkim wypełnieniem) lub Bro (flat color)
  • Primary: #4a8d7a (zielony marki)
  • Secondary: #1c1a18 (ciemny ink)
  • Background: transparent
  • Eksport: SVG

⚠️ Ważne uwagi

  • Atrybucja: Storyset darmowy = link w stopce. Pro (~$12/mies) = bez atrybucji.
  • Spójność stylu: wszystkie ilustracje z jednego stylu Storyset — nie mieszać.
  • Optymalizacja: przepuść SVG przez SVGOMG — redukcja 30-60%.
  • Komponent działa bez SVG — ma fallback emoji + gradient. Możesz wdrożyć paczkę zanim zdobędziesz wszystkie ilustracje.
